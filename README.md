# Chicken-Arena-Game
A 3D party game built in Python and Panda3D, inspired by the roosters at my home in Sanghar, Sindh. Control a rooster collecting eggs against 3 AI opponents using vectorised NumPy scoring, reaction lag, and sprint bursts. Rooster modelled in Blender, textured in Photoshop. 120-second matches, custom HUD timer, rigged GLB actors.
